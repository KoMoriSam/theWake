## 开张了，但没完全开张

> 2026 年 6 月 24 日，个人博客堂堂上线！

---

说实话，这句话我本来打算在 2023 年就写出来的。

从手搓原生三剑客 `HTML` + `JS` + `CSS` 开始，到接触 `Vite` + `Vue 3`，从零散几篇 Markdown 滚到三个独立仓库。

直到 AI LLM 的飞速发展，Vibe coding 像天降神兵一样，把我从非科班出身、非要写代码的泥潭里捞出来，这个项目才终于从半成品变成了大概能用的成品。

这篇推文不是博客的第一篇文章，我正在把过去几年零零碎碎写的东西整理进来（依旧「锐意制作中……」）。

---

## 关于我

## 三个仓库，一个站点

我的整个写作系统（并非系统）由三个仓库组成：

### 📝 [theWake](https://github.com/KoMoriSam/theWake) —— 博客文章源

这里放的是我所有非虚构类的文字：

- **随笔 & 读书笔记**：看到什么、想到什么，就随手（也没有那么随手）记下来。
- **想法碎片**：那些不够成篇、但值得被保存的灵光一现（大概率也不会记录下来……）。

> 这里大部分东西都挺个人的——没有干货，没有“教你如何 XXX”。如果你愿意读，我很感激；如果你觉得矫情，我也完全理解。

### 📖 [theHorizon](https://github.com/KoMoriSam/theHorizon) —— 长篇小说源

书名暂定 **《向远方》** ，主角叫方远洛。

目前写得非常零碎，没有完整的大纲，没有具体的规划，纯纯“赛博露阴癖”——把聊天记录、朋友圈截图、深夜对话框里的胡话统统塞进小说里，试图拼凑出一种属于这个时代的叙事方式。

> 可爱美丽善良大方的女主角林瑾禾（尚未出场）正在赶来的路上。是的，我知道这听起来很像自嗨，但……把“自己”写进小说这种事情，难道不是每个写过字的人都偷偷想过吗？（也许吧。）

### 🌐 [komorisam.github.io](https://github.com/KoMoriSam/komorisam.github.io) —— 站点本身

技术栈是 **Vite + Vue 3 + Tailwind CSS + daisyUI**，已经迭代到 `v1.11.1`。

它支持：

- 亮色 / 暗色 / 跟随系统主题
- 阅读进度条（让你知道还有多久能看完我的废话）
- 段落评论（基于 Giscus，欢迎在任意段落留下你的想法）
- 多种中文字体切换（满足一点点排版强迫症『还是有很多排版 bug 的』）

从一个只有首页的简陋小站到现在勉强算完整的博客，中间经历了无数次推翻重来。Git 提交记录甚至是最近几天才规范化的，里面写满了毫无识别性的提交记录、`fix: 修 bug`、`feat: 加功能`，以及若干条 `chore: 重写、移路径、刷版本号……`。

## 所以，这个博客用来做什么？

没有明确的定位，没有更新计划，没有 KPI。

它就是一个我存放文字的地方。

那些在社交媒体上写不下、在日记本里写不完、在工作文档里写不出的东西，我会试着放在这里。可能是一篇 3000 字的读书札记，可能是一段 300 字的深夜牢骚，也可能是某个小说章节的草稿。

如果你偶然路过，读到了某句话让你觉得“啊，原来也有人这么想”，那这个博客就算没有白开。

## 接下来呢？

谁还没几个写了就倒的 flag 呢？趁着开张，我决定把它们都列出来。

### 网站方面

- [ ] **完善博客的 SEO**（现在网页搜索「KoMoriSam」出不来文章捏）
- [ ] **完善搜索功能**（现在简单搜索能用就先用着吧，我只是给个愿景）
- [ ] **做个书架**（放我最近在读的书，以及想读但一直没翻开的）
- [ ] **性能与可访问性**（只能说是全靠 AI 瞎写的后果，没有控制打包体积，很多性能问题也没优化，现在调试网页的时候卡到爆炸）
- [ ] **总目标**：今年内把版本号推到 `v2.0.0` 有感觉吗？（并非刷版本号）

### 写作方面

- [ ] **写完小说大纲**：林瑾禾必须堂堂出场！（我保证）
- [ ] **争取年底前写完第一卷**：如果写不完……那就写不完吧（汗）
- [ ] **开个短篇坑**：用博客连载一个 2～3 万字的短篇小说，题材还没想好，可能是悬疑，也可能是日常。反正写不完就赖给 AI（不是）。
- [ ] **保持周更**：至少每周发一篇东西，哪怕是读书笔记或几百字的随想。立 flag 的时候总是很轻松，就看能坚持几周了。

### 生活方面

- [ ] **多读几本书**：「我也劝同志们多读一点书」
- [ ] **学点新东西**：没想好具体的，有一点是肯定的——继续学编程（或者说：AI 驯服术这一块）
- [ ] **走出去**：多拍点照片，多记点对话，给写作攒点真实素材。

计划就是这么些计划，flag 就是这么些 flag。  
能不能全部实现？嗯……可能吧……
